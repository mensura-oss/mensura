# Mensura — Development Roadmap

## Phase 0 — Foundation

- Create monorepo.
- Define package boundaries.
- Set up CI.
- Create design system.
- Establish coding standards.
- Publish governance docs.

## Phase 1 — MVP desktop core

- Tauri shell.
- React UI skeleton.
- Monaco editor integration.
- xterm.js terminal integration.
- Workspace/project creation.
- Local Git repo connection.
- Chat panel.
- Single-agent task run.
- Run logs and diff display.

## Phase 2 — Core orchestration

- FastAPI Core service.
- LangGraph workflow execution[cite:72][cite:79][cite:85].
- Provider abstraction.
- BYOK settings.
- Retry and checkpoint logic.
- Human approval gates.

## Phase 3 — Vault

- File/document ingestion.
- Embeddings.
- Semantic search.
- Architecture summary generation.
- Linked task memory.

## Phase 4 — Guard

- Lint/test integration.
- Secret scanning.
- Risk scoring.
- Protected path policies.
- Review required for risky changes.

## Phase 5 — Team mode

- Shared Core deployment.
- Team workspaces.
- Shared memory.
- User roles.
- Audit trail UI.

## Phase 6 — Hub ecosystem

- Plugin manifest format.
- Local plugin installation.
- MCP connector gallery.
- Agent template packs.

## Phase 7 — Advanced features

- Multi-agent swarm mode.
- Graph memory UI.
- Voice commands.
- Scheduled runs.
- Remote job runners.
- Release agent.
